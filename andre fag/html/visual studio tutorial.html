Hvordan laste ned Visual Studio Code og åpne en live server

Visual Studio Code er et program som brukes for å programmere. Men som på nesten alle program, starter reisen med å laste ned programmet.



Steg 1

først vil du søke opp "Visual Studio Code", eller klikke på denne lenken Visual Studio Code

Når du er på nettsiden, vil du laste ned programmet. Hvis du er på windows, trykk på "Download For Windows" knappen, eller velg alternativene for Mac eller Linux.

Steg 2

Nå vil visual code setuppen lastes ned. Åpne deretter denne og følg gif-en.

Steg 3

Etter Visual Studio har lastet ned, kan du enten bruke desktop ikonet eller søke den opp i søkefeltet for å åpne programmet.

Når du har åpnet Visual Studio Code, vil du bli møtt med fargevalg. Velg den du liker mest, så gå videre til "tilbake"

Nå kan du begynne å lage din egne første nettside, men først må du laste ned en Live Server extention. Følg gif-en

Steg 4

Når live server er installert, skal du lage en folder og en fil. Du kan følge videoen, men husk at filen skal slutte med ".html"

Steg 5

nå skal du lage en funksjonell nettside. Du starter med å skrive "!", så enter. Inni body taggen vil kan du skrive og gjøre hva du vil, som å legge til et paragraph som sier "hallo verden!"

Så høyre klikker du og åpner en Live Server.
